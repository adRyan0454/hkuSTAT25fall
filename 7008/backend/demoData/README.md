# 测试数据生成工具

## 📁 文件夹说明

此文件夹包含用于生成测试数据的脚本，**仅用于开发和测试环境**。

在生产环境中，这些脚本不会被使用。

---

## 📋 文件列表

### 1. `generate_full_data.py`
完整数据生成脚本

**功能**：
- 生成155名员工（5个部门 × 31人）
- 建立完整的管理层级关系（P5→P4→P3→P2→P1）
- 生成12个月的绩效数据（2024年11月-2025年10月）
- 总计：1,860条绩效记录

**使用方法**：
```bash
python backend\dataTest\generate_full_data.py
```

### 2. `reset_and_init.py`
数据库重置脚本

**功能**：
- 清空所有员工和绩效数据
- 保留管理员账号
- 重新初始化5个部门

**使用方法**：
```bash
python backend\dataTest\reset_and_init.py
```

- 每个员工都有明确的上级

### 4. 职级差异

## ⚠️ 注意事项

### 1. 仅用于测试
- 这些脚本仅用于开发和测试环境
- 生产环境不应使用这些脚本
- 数据是随机生成的，不代表真实业务

### 2. 数据量
- 155名员工 × 12个月 = 1,860条记录
- 生成时间约15-40秒
- 数据库大小约8-15MB

### 3. 重新生成
如果需要重新生成数据：
1. 运行 `reset_and_init.py` 清空数据
2. 运行 `generate_full_data.py` 生成新数据
3. 重启后端服务
